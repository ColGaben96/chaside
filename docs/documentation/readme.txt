El archivo ejecutable está localizado en ./release/loquesea.jar <-- Porfavor nota que ./ significa que la localizacion está en donde sea que hayas exportado el .zip. Si exportaste el .zip en la carpeta de descargas, tienes que encontrar la carpeta %userprofile%\Descargas\CHASIDE\release.

===================================================================================================
					HOW TO RUN
===================================================================================================

WINDOWS

via console:

Open CMD or PowerShell and do:

cd location_of_CHASIDE\CHASIDE\release\

java -jar chaside.jar

via GUI:

Find the folder of CHASIDE
Find release folder
double click chaside.jar

LINUX AND MACOS

via console:

cd location_of_CHASIDE\CHASIDE\release\

java -jar chaside.jar

via GUI:

If you're using linux, use your file explorer of preference and find the .jar file
If you're using MacOS, use finder, and do the same of Windows.
double click loquesea.jar

If you're using MacOS and don't know how to double click in a runnable file, call Tom, your butler to do it for you.
